# certified.cert_base — FullCert

`FullCert` is the base class for all certificate types that hold both a certificate
and its corresponding private key. `CA` and `LeafCert` (see [ca.md](ca.md)) both
extend this class.

Originally derived from [trustme](https://github.com/python-trio/trustme) (MIT license).
